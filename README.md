# CASA BLANCA BAR & RESTAURANT - ADMIN


## Prerequisites

Make sure you have the following installed on your machine:

- [Node.js](https://nodejs.org/) (v14 or newer)
- [Yarn](https://yarnpkg.com/)

## Getting Started

1. **Clone the repository:**

   ```bash
   https://github.com/ipetersenpai/CasaBlanca-Admin
   ```


2. **Navigate to the project folder:**

   ```bash
   cd casablanca-admin
   ```

3. **Install dependencies:**

   ```bash
   yarn install
   ```

4. **Start the development server:**

   ```bash
   yarn dev
   ```

   This will start the development server, and you can view your app at `http://localhost:3535`.

## Build for Production

To build the app for production, run:

```bash
yarn build
```

This will generate a `dist` folder with the optimized production build.

## Additional Commands

- **Lint your code:**

  ```bash
  yarn lint
  ```

- **Run tests:**

  ```bash
  yarn test
  ```
